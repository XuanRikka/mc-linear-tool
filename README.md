# mc-linear-tool

## 安装方法

### 方法一（推荐）
```bash
cargo install --git https://github.com/mccxj/mc-linear-tool.git
```

### 方法二
下载Release中发布好的预构建包


## 使用说明
具体使用方法见cli的--help
以下是注意事项：
 - anvil的lz4支持未经测试
 - linear v1因已过时而不建议使用
